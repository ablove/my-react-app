# To-Do List App (React + TypeScript)

This is a simple To-Do List application built using React and TypeScript. The app allows users to create, edit, delete, and mark tasks as completed.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed

## Technologies Used

- React
- TypeScript
- CSS (for styling)
